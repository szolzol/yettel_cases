Fraud Detection SaaS

Real-time fraud detection across SIM swap, subscription fraud, IRSF, and revenue assurance.

Shortlisted VendorsSubex HyperSense (AI-first, end-to-end FMS) · Mobileum AIP (voice/roaming/signaling focus) · NVIDIA Morpheus (GPU-accelerated custom build)
Deployment ModelSubex/Mobileum: SaaS with per-subscriber licensing · Morpheus: on-prem GPU infrastructure
Est. Annual Cost€300K–500K/yr (SaaS) · €800K–1.2M/yr (Morpheus incl. GPU infra + ML team)
Implementation6–8 weeks (Subex/Mobileum turnkey) · 4–6 months (Morpheus custom build)
Key DifferentiatorSubex leads with 200+ pre-built telecom fraud rules and closed-loop automation. Mobileum strongest on voice/roaming fraud. Morpheus only if building proprietary real-time engine.
Regional SupportMobileum operates regional hubs with direct CEE support. Subex has European presence but relies on remote implementation.
RecommendationSubex HyperSense — fastest time-to-value, deepest telecom domain coverage, SaaS model avoids GPU capex. Reserve Morpheus as Year 2 option if Yettel builds internal ML team.

Negotiation Framework

📋 IP Ownership Clauses
Yettel retains ownership of all models trained on proprietary data. Vendor provides model explainability documentation. Source code escrow required for critical dependencies. Audit rights for AI decision-making processes.
🔐 Data-Sharing Terms
Data residency within EU (GDPR compliance). No vendor use of Yettel data for model training without explicit consent. Data deletion upon contract termination within 30 days. Regular data access audits.
🚪 Exit & Portability Clauses
12-month maximum lock-in period. Full data export in standard formats within 60 days. Transition support for 6 months post-termination. No penalty for early exit after initial term. API compatibility guarantee for migration.
